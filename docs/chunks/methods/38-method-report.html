Get report status

GET https://mapon.com/api/v1/report/status.json
  • Description
  • Parameters
  • Response
Check if report is ready for download.
Possible statuses:
  • queued - waiting in line to be processed
  • processing - building report
  • finished - report is available for download
key *
string
API Key
id *
int
Make process ID
{
    "data": {
        "status": "finished",
        "process_id": "1",
        "file_name": "report.xlsx",
        "file_url": "https://mapon.com/api/v1\/file.php?1"
    }
}

Get Ely Report

GET https://mapon.com/api/v1/report/ely.json
  • Description
  • Parameters
  • Response
  • Error codes
Get Ely Report data
key *
string
API Key
date_from *
string
Date of period start. Format Y-m-d
date_to *
string
Date of period end. Format Y-m-d
time_from
string
Time of period start. Format H:i. Optional parameter. By default set to 00:00
time_till
string
Time of period start. Format H:i. Optional parameter. By default set to 23:59
unit_id *
int
Car id for required period.
page
int
Page number.
limit
int
Number of items per page. Minimum amount is 10 items.
enabled_columns
string
Coma separated list. For example: step,lat,lng
date - Date. Enabled by default.
time - Time. Enabled by default.
step - Step
lat - Latitude
lng - Longitude
temp_left - Temp left (°C)
temp_mid - Temp mid (°C)
temp_right - Temp right (°C)
temp_reserve - Temp reserve (°C)
screed_width - Screed width (m)
screed_selector - Screed selector
screed_left_volt - Screed left (mV)
screed_right_volt - Screed right (mV)
ambient_temp - Ambient Temperature (°C)
ambient_humidity - Ambient Humidity (%)
screed_left - Screed left (mm)
screed_right - Screed right (mm)
next_cursor
string
Date time GMT
Response consists of columns, rows, nextCursor:
columns - Selected columns,
rows - Data according to selected columns,
nextCursor - Date time of the item in the next page.
