Insert/Update menu item

POST https://mapon.com/api/v1/application_menu/save.json
  • Description
  • Parameters
  • Response
  • Error codes

Insert/Update menu item. Edited menu items will be available after 5 minutes frontend cache expire.

You can also update first level menu item by passing parent_id = 0.

Iframe url will be formatted after this structure app_base_url + path + ?user_token=

key *
string
API Key
id
int
Menu item ID. If not empty, menu item with this ID will be updated
parent_id *
int
Parent menu item ID
title *
string
Menu item title
path
string
Menu item path. Will be appended to iframe src URL. Should start with /.
{
    "data": {
        "status": "ok"
    }
}
CodeDescription
1Application not found
2Menu item not found
3Invalid parent_id parameter
4Invalid title parameter
