List units driver behaviour data
key *
string
API Key
date_from *
string
Formatted date Y-m-d. e.g. 1970-01-01
date_till *
string
Formatted date Y-m-d. e.g. 1970-01-01
depot_id
int
Depot id
group_id
int
Group id
unit_ids
string
Comma separated unit ids
metrics
string
Comma separated metrics. Set metrics that you do want them to be taken into account when calculating driving results.
Allowed values: effective_speed,excessive_idling, cruise_control,coasting,harsh_braking, harsh_acceleration,harsh_cornering,green_rpm,speeding
By default enabled all.
page
int
Page no. Starts with 1
per_page
int
Items count per page. Default: 1000
Distance is returned in km, duration seconds. All fields except unit_id, title may be null.
Please note that if fuel_avg_consumption_measure value is kg/100km then all fuel values are returned in KG, not in L
