Create a preset

POST https://mapon.com/api/v1/preset/create.json
  • Description
  • Parameters
  • Response
Creates a new preset
key *
string
API Key
title *
string
Preset title.
has_access_to_all_cars
bool
Does preset allow access to all cars.
has_access_to_all_drivers
bool
Does preset allow access to all drivers.
has_access_to_all_objects
bool
Does preset allow access to all objects.
has_access_to_all_menus
bool
Does preset allow access to all menus.
select_car_depots
bool
Restrict preset vehicle permissions depending on selected depots.
select_driver_depots
bool
Restrict preset driver permissions depending on selected depots.
has_access_to_all_ble_beacons
bool
Does preset allow view access to all BLE beacons
menu_ids *
int[]
Array of menu ids.
car_ids
int[]
Array of car ids.
car_group_ids
int[]
Array of car group ids. Only first level ID's needed.
car_depot_ids
int[]
Array of depot ids. Required if select_car_depots is set to true
driver_ids
int[]
Array of driver ids.
driver_depot_ids
int[]
Array of depot ids. Required if select_driver_depots is set to true
object_ids
int[]
Array of object ids.
object_group_ids
int[]
Array of object group ids.
ble_beacon_ids
int[]
Array of BLE beacon ids.
See options via BLE beacon list endpoint
permissions
bool[]
Array of permissions, key -> value.
See options via Available preset permissions endpoint
restrictions
bool[]
Consists of:
restricted_mobile_access - boolean, whether to restrict access to system from mobile phone
restricted_ip_range - string, range of restricted IP's
custom
integer
Whether or not preset is customized for a single user. Custom preset can only be assigned to 1 user. Values - 1 or 0
{
    "id": 123,
    "status": "successful"
}

Modify an existing preset

  • Description
  • Parameters
  • Response

If arrays with ID's (fields that are suffixed with _ids) are not passed, those are considered to be deleted.
To keep the array information present, mirror data from Preset data endpoint.
