Edit

POST https://mapon.com/api/v1/tracking/edit.json
  • Description
  • Parameters
  • Response
Edit existing tracking link
key *
string
API Key
id *
int
Tracking link unique ID
from *
int
Tracking available from (Datetime timestamp)
till *
int
Tracking available till (Datetime timestamp)
routes *
int
0 to disable or 1 to allow see routes in period
car_id *
[]
Integers of car Ids
restrict_routes_period
[]
Additional parameters
time_from - Specific time from (Example: 09:00)
time_to - Specific time to (Example: 15:00)
At least one of these keys must be defined:
d1
d2
d3
d4
d5
d6
d7
comment
string
Comment for tracking link
comment_visibility
int
Make tracking link comment visible on tracking page. 0 - not visible, 1 - visible. Default value is 0
language
string
Tracking link language
deactive_object_id
int
When car will arrive in the territory it will be removed from tracking link
active_in_object_id
int
Tracking activates only if vehicle is in defined territory
additional_data
[]
Additional data for carrier
carrier_temp - []
Example:
carrier_temp":[{"comp_id":1,"types":["supply","defrost","return","setpoint"]}
