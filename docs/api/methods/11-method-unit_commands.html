Get unit available command list
key *
string
API Key
unit_id *
int
unit id
Returns empty array if no commands available.
Full list of available commands:
  • open_doors
  • close_doors
  • open_trunk
  • open_windows
  • close_windows
  • hazard_lights
{
    "data": [
        "open_doors",
        "close_doors",
        "open_trunk",
        "open_windows",
        "close_windows",
        "hazard_lights"
    ]
}
