Change password

POST https://mapon.com/api/v1/user/change_password.json
  • Description
  • Parameters
  • Response
  • Error codes
Changes existing users password
key *
string
API Key
user_id *
integer
Unique user ID
password *
string
New password for the user. Must be at least 8 characters long, must contain at least 1 number, must contain at least 1 letter and/or symbol. Must be different than the current users password.
force_change
integer
If set, will force to ignore password validation rules. The password still cannot be empty.
{
    "status": "successful"
}
CodeDescription
1Invalid or unknown user_id parameter
2Password does not fulfill complexity requirements. Accompanied by a message that explains which rule check failed
3Password is the same as the current users password
4No access to user with id user_id
