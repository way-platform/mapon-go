Fuel sensor level data

GET https://mapon.com/api/v1/fuel/data.json
  • Description
  • Parameters
  • Response
  • Error codes
Returns fuel sensor levels for selected period
key *
string
API Key
unit_id *
int
One unit ID
from *
datetime
Time from which to gather data
till *
datetime
Time until which to gather data
data_source
string or []
Include data entries for:
sensor - fuel levels from fuel rods
can - fuel levels from board computer
flow - flow rate in time segment
If no specific data_source given, it will return all installed sources
Please note that if unit's fuel consumption measurement is kg/100km, then all fuel values are represented in KG, not in L
